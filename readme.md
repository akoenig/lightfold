# lightfold

A lightbox component using [@reduct](https://github.com/reduct) framework.

## Installation

```sh
npm install
```

You have to serve this directory from a web server. Consider [serve](https://www.npmjs.com/package/serve).

Spin up your browser. Enjoy :D
